# FH6Auto SArB1e 深度优化版 v1.2.3

## 本次修复与改进
- 优化移除车辆逻辑，默认继续识别可删模板并排除全新标签，降低误删风险。
- 新增默认关闭的“快速删除筛选结果”选项，适合用户确认筛选结果干净后跳过车辆模板识别。
- 初步修复 issue #1 提及的删车选择罕见问题，删车流程保留车辆操作菜单确认，避免点击车辆后误触发上车。

## 使用提醒
- 请先提前点赞车辆调教和蓝图。
- 跑图车辆建议使用默认涂装。
- 游戏难度建议改为【所向披靡】。
- 建议桌面分辨率使用 2K。
- 脚本涉及自动化操作，请在确认车辆、筛选条件和流程设置无误后再启动。

## 下载说明
- 请下载本 Release 附件中的 `FH6AutoStableVersion.By.SArB1e.exe`。
- `dist/` 目录下的 EXE 仅作为 Release 附件发布，不提交到源码仓库。
