# FH6Auto SArB1e 深度优化版 v1.2.3

## 本次修复与改进
- 新增“冬季雪地胎筛选优化”开关，适配赛季筛选面板新增雪地胎选项后的菜单位置变化。
- 启用该开关后，超级抽奖选车和移除车辆在按 Y 打开筛选面板后，会额外下移 2 格再勾选“重复项”。
- 默认关闭该优化，非冬季或筛选菜单未变化时仍沿用原有固定筛选路径。

## 使用提醒
- 请先提前点赞车辆调教和蓝图。
- 跑图车辆建议使用默认涂装。
- 游戏难度建议改为【所向披靡】。
- 建议桌面分辨率使用 2K。
- 脚本涉及自动化操作，请在确认车辆、筛选条件和流程设置无误后再启动。

## 下载说明
- 请下载本 Release 附件中的 `FH6AutoStableVersion.By.SArB1e.exe`。
- `dist/` 目录下的 EXE 仅作为 Release 附件发布，不提交到源码仓库。
