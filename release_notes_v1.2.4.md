# FH6Auto SArB1e 深度优化版 v1.2.4

## 本次修复与改进
- 修复 CR 识别失败时买车流程被误阻断的问题：当 CR 数值无法识别时，跳过余额检查继续买车（与 v1.1.0 行为一致），不再因识别问题导致整个买车模块失败。

## 使用提醒
- 请先提前点赞车辆调教和蓝图。
- 跑图车辆建议使用默认涂装。
- 游戏难度建议改为【所向披靡】。
- 建议桌面分辨率使用 2K。
- 脚本涉及自动化操作，请在确认车辆、筛选条件和流程设置无误后再启动。

## 下载说明
- 请下载本 Release 附件中的 `FH6AutoStableVersion.By.SArB1e.exe`。
- `dist/` 目录下的 EXE 仅作为 Release 附件发布，不提交到源码仓库。
